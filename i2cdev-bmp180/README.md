BMP180
====

An I2C driver for the BMP180 barometer, thermometer.

## Usage
Add the following to your `Cargo.toml`:
```toml
[dependencies]
i2csensors = "0.1.*"
i2cdev-bmp280 = "0.1.*"
i2cdev = "0.3.1"
```


i2cdev-bmp180 = "0.1.*"
i2cdev-l3gd20 = "0.1.*"
i2cdev-lsm303d = "0.1.*"
